CSS syntaxis

Een stylesheet bestaat uit selectors en declaration blocks.
Een declaration block bestaat uit declarations (of stijlregels).
Stijlregels bestaan uit properties & values.

CSS properties

We zijn daarbij de volgende properties tegengekomen:

  • background-color
  • color
  • font-size
  • font-family
  • font-weight
  • font-style
  • text-decoration
  • list-style-type
